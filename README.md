# 基于Layuiv2.2.45做的后台模板  
作者:[Dust](http://dust101.lofter.com)  
-----
## 特色:
 * 基于Layuiv2.2.45，绝不同于百度搜索到的同类Layui后台模板的layuiv1版本
 * 单页面 Tab标签式
 * 可以从收藏夹里打开指定的Tab
 * Tab切换或打开带有渐变的视觉特效
 * 简单，快捷，复制index.html出来引用Layui即可。
 * 以后更新...
